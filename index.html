Storm

A probabilistic model checker

Storm is a modern model checker for models featuring randomness. It has been designed with performance and modularity in mind

Running Storm

Storm has been written in native C++, which allows to use high-performance libraries for numerical and algebraic tasks. The source code is freely available. For users that just want to peek at storm, we provide a virtual disk image.

Prism

The Prism language is a command language to specify Markov models.

JANI

JANI is an interchange format for Markov Models, based on a straightforward JSON representation of networks of quantitative automata with variables. Discrete variables and a parallel composition operation allow models with large or even infinite state spaces to be represented succinctly.

GSPN

Generalised Stochastic Petri Nets (GSPNs) extend Petri nets with random timing and allow to resolve conflicts probabilistically.

DFT

Dynamic Fault Trees (DFTs) are an expressive and concise formalism to express how component failures lead to system failures. Several aspects of DFTs have been standardised.

cpGCL

cpGCL is an probabilistic extension of Dijkstra's guarded command language. Probabilistic programming languages are an implementation-near model for several algorithms featuring randomisation.
